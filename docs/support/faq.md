# 常见问题（FAQ）

## NovaAPI 和直接用 OpenAI 有什么区别？

NovaAPI 用一套 OpenAI 兼容接口聚合了 40+ 上游供应商，一把 Key 即可调用多家模型，并统一计量计费、容灾与限流。

## 如何切换模型？

只需修改请求里的 `model` 字段，其余代码不变。可用模型见[模型列表](/api/models)。

## 支持流式输出吗？

支持。在请求中设置 `"stream": true`，详见[对话补全](/api/chat)。

## 余额可以退款吗？

由于 API 服务的特殊性，充值成功后余额原则上不退款，但可永久用于平台任何服务消费。详见服务协议。

## 调用报错怎么排查？

先看返回的 HTTP 状态码与错误信息，对照[错误码](/support/errors)。常见为密钥无效（401）或触发限流（429）。
