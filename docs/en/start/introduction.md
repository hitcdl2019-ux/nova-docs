# Introduction

NovaAPI is an **LLM aggregation platform**. With a single OpenAI-compatible API, it lets you access 40+ mainstream model providers through one gateway.

## What it does

- **Unified access**: one `base_url` + one API Key to call OpenAI, Claude, Gemini, and many more.
- **Transparent metering & billing**: pay-as-you-go, with usage, quota, and call logs in one console.
- **Stable & highly available**: multi-upstream failover and rate control, ready for production.
