# 平台简介

NovaAPI 是一个**大模型聚合平台**，用一套 OpenAI 兼容接口，帮你一站式接入 40+ 主流大模型供应商。

## 能做什么

- **统一接入**：一个 `base_url` + 一把 API Key，调用 OpenAI、Claude、Gemini 等众多模型。
- **计量计费透明**：按量计费，控制台统一查看用量、额度与调用日志。
- **稳定高可用**：多上游容灾与速率控制，适配生产环境。
